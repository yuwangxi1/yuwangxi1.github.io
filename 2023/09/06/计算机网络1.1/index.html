第一节：计算机网络概述

本节导语：需要记忆内容简单并且量小，计算题也相对简单，属于非重点章节（多记忆即可，并不需要太多思考）

计算机网络的概念

计算机网络是一个将分散的、具有独立功能的计算机系统，通过通信设备和线路连接起来，由功能完善的软件实现资源共享和信息传递的系统。简而言之，计算机网络就是一些互连的、自治的计算机系统的集合。

互连：通过链路互联互通 自治：无主从关系

计算机网络可以被看作一整个蜘蛛网，其中丝线的交点为计算机系统，通过这些通信设备和线路进行连接，最后由一些完善的软件实现数据通信。

计算机网络的组成

从不同的角度，可以将计算机网络分为许多类：

  • 组成部分

    由 硬件、软件、协议组成

    硬件主要由主机（也称端系统）、通信链路（如双绞线、光纤）、交换设备（如路由器、交换机等）和通信处理器（如网卡）等组成。

    软件主要包括各种实现资源共享的软件和方便用户使用的各种工具软件（如网络操作系统、邮件收发程序、FTP程序、聊天程序等）。软件部分多属于应用层。

    协议是计算机网络的核心，如同交通规则制约汽车驾驶一样，协议规定了网络传输数据时所遵循的规范。

  • 工作方式

    分为边缘部分和核心部分

    边缘部分由所有连接到因特网上、供用户直接使用的主机组成，用来进行通信（如传输数据、音频和视频）和资源共享；

    核心部分由大量的网络和连接这些网络的路由器组成，它为边缘部分提供连通性和交换服务；

    我们作为服务使用方，能够接触到的就是主机，即边缘部分；相应的为我们服务的，就是核心部分

  • 功能组成

    由 通信子网 和 资源子网组成

    通信子网由各种输出介质、通信设备和对应的网络协议组成，它使网络具有数据传输、交换、控制和存储的能力，实现联网计算机之间的数据通信。

    资源子网是实现资源共享功能的设备以及其软件的集合，向网络用户提供共享其他计算机上的硬件资源、软件资源和数据资源的服务。

    资源子网对应OSI模型中的上三层：应用层、表示层、会话层

    通信子网对应OSI模型中的下三层：网络层、数据链路层、物理层

计算机网络的功能

五大主要功能

  • 数据通信

    数据通信是计算机网络最基本和最重要的功能，用来实现联网计算机之间各种信息的传输，并将分散在不同地理位置的计算机联系起来，进行统一的调配、控制和管理。例如：文件传输、电子邮件等应用，离开了网络将无法实现。

  • 资源共享

    资源共享可以是软件共享、数据共享，也可以是硬件共享。它使计算机中的资源互通、分工协作，从而极大地提高硬件资源、软件资源和数据资源的利用率。

  • 分布式处理

  • 提高可靠性

  • 负载均衡

计算机网络的分类

六种分类方式：

  • 分布范围

    -广域网（WAN）

    广域网是因特网的核心部分。广域网的任务是提供长距离通信，运送主机所发送的数据，其覆盖的范围通常是直径为几十千米到几千千米，因而有时也称远程网。连接广域网的各节点交换机的链路一般都是高速链路。传统上，广域网采用交换技术。

    -城域网（MAN）

    城域网大多采用以太网技术。

    -局域网（LAN）

    局域网一般用微机或工作站通过高速线路相连，覆盖范围较小，通常是几十米到几千米的区域。局域网在计算机配置的数量上没有太多限制。传统上，局域网使用广播技术。

    局域网可分为三大类：1、平时常说的局域网LAN。2、采用电路交换技术的局域网，称计算机交换机CBX（Computer Branch eXchange）或PBX（Private Branch eXchange）。3、新发展的高速局域网HSLN（High Speed Local Network）。

    -个人区域网（PAN）

    个人区域网一般采用无线技术，只涉及个人工作的地方，将手机、平板之类用无线技术连接起来的网络，也常称为无限个人区域网（WPAN）

    注意：若中央处理器（CPU）之间的距离非常近（如仅1m的数量级或甚至更小），则一般称为多处理器系统，而不称为计算机网络

    现在大量的计算机是通过诸如以太网这样的局域网连入广域网的，而局域网与广域网的互连是通过路由器实现的。

  • 传输技术

    -广播式网络

    -点对点网络

    是否采用分组存储转发与路由选择机制是点对点式网络与广播式网络的重要区别，广域网基本属于点对点式网络

  • 拓扑结构

    网络拓扑结构是指由网中结点（路由器、主机等）与通信线路（网线）之间的几何关系（如总线形、环形）表示的网络结构，主要指通信子网的拓扑结构。

    -总线形

    -星形

    -环形

    -网状

    星形、总线形和环形网络多用于局域网，网状网络多用于广域网

  • 使用者

    -共用网

    -专用网

  • 交换技术

    -电路交换网络

    -报文交换网络

    -分组交换网络

  • 传输介质

    -有线网络

    -无线网络

计算机网络的性能指标

  • 带宽

    在计算机网络中，带宽表示网络的通信线路所能传送数据的能力，是数字信道所能传送的“最高数据传输速率”的同义词，单位是比特/秒（b/s）。

  • 时延

    发送时延

    发送时延也叫传输时延。结点将分组的所有比特推向（传输）链路所需的时间，即从发送分组的第一个比特算起，到该分组的最后一个比特发送完毕所需的时间。

    计算公式：发送时延 = 分组长度 / 信道宽度 = 数据长度 / 信道带宽（发送速率）（数据传输速率）

    传播时延

    电磁波在信道中传播一定的距离需要花费的时间，即一个比特从链路的一端传播到另一端所需的时间。

    计算公式为：传播时延 = 信道长度 / 电磁波在信道上的传播速率

    处理时延

    数据在交换结点时为存储转发而进行的一些必要的处理所花费的时间。例如，分析分组的首部、从分组中提取数据部分、进行差错检验或查找适合的路由等。

    排队时延

    分组在进入路由器后要先在输入队列中排队等待处理。路由器确定转发端口后，还要在输出队列中排队等待转发，这就产生了排队时延。

​ 对于高速链路，提高的仅是数据发送速率而非比特在链路上的传播速率。提高数据的发送速率只是为了减少数据的发送时延

  • 时延带宽积

    时延带宽积 = 传播时延 * 信道带宽

  • 往返时延（RTT）

  • 吞吐量

    指单位时间内通过某个网络（或信道、接口）的数据量。

  • 速率

    网络中的速率是指连接到计算机网络上的主机在数字信道上传送数据的速率，也称数据传输速率、数据率或比特率，单位为b/s（比特/秒）（有时也写bps）。数据较高时，可用kb/s(k = 10^3)、Mb/s(M = 10^6)或Gb/s(G = 10^9)表示。在计算机网络中，通常把最高数据传输速率叫做带宽。

    这里的单位要和内存中的单位分开看内存中单位做区分1B = 8 bit 1KB = 2^10B 1MB = 2^10KB 1GB = 2^10MB

  • 信道利用率

    信道利用率 = 有数据通过的时间 / （有 + 无）数据通过的时间。

课后题：

有两个网络，它们都提供可靠的面向连接的服务，一个提供可靠的字节流，另一个提供可靠的报文流。请问二者是否相同？为什么？

答案：
不相同。在报文流中，网络保持对报文边界的追踪；而在字节流中，网络不进行这样的追踪。例如，一个进程向一条连接写了1024B，稍后又写了1024B，那么接收方共读了2048B。对于报文流，接收方将得到两个报文，每个报文1024B。而对于字节流，报文边界不被识别，接收方将全部1024B作为整体，在此已经体现不出原先有两个不同报文的事实。
